markov.py contains a basic framework for generating markov chains from plaintext. It's efficient and relatively successful.
jokebot.py accesses reddit's API to retrieve jokes from r/jokes and attempt to create original material. It's not very funny.

Features:
	.> Fast handling of large amounts of text data, along with basic analysis of the document
	.> Use of reddit's API to collect new material
	.> Proves conclusively that we are in no danger of being exterminated by self-aware machines

Future additions:
	.> Examine for punctuation, capitalization to give greater grammatical consistency
	.> Dynamically change fragment length based on punctuation
